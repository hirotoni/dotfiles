# dotfiles

Personal dotfiles for macOS. Managed with Homebrew + GNU Stow.

## Setup

```bash
git clone git@github.com:hirotoni/dotfiles.git ~/dotfiles
cd ~/dotfiles

# run with target
./install.sh zsh
./install.sh nvim
```

## Machine-specific config

```bash
cp zsh/.zsh/local.zsh.example zsh/.zsh/local.zsh
```
